# Main-Portfolio


This is my personal portfolio website, built to showcase my projects, skills, and experience as a junior web developer.
The site was created with a focus on clean design, responsiveness, and accessibility. It reflects my learning journey in web development and highlights both my technical abilities and my approach to building user-friendly interfaces.
Features -
Responsive design for mobile, tablet, and desktop
Projects section showcasing real-world builds
Skills section highlighting front-end and back-end technologies
Clean, minimal UI focused on usability
Accessible and semantic markup
